# Programs

Named initiatives ShikshaLokam runs or convenes.

## Active

### Shikshagraha
The people's movement; ShikshaLokam catalyses, does not own. "A people's movement to improve India's 1 million public schools." Anchored in Sense → Make Sense → Learn → Improve. Restores agency of system leaders, communities, youth, women.
Source: [shikshalokam.org](https://shikshalokam.org/)

### InvokED — A Global Dialogue on Education Leadership
Annual convening.
- **InvokED 2026:** held 6–7 February 2026 at Prestige Centre for Performing Arts (PCPA), Bengaluru. Speakers included SD Shibulal (SFPI / ShikshaLokam), Wendy Kopp (Teach For All), Martin Kasper (Childaid Network Foundation), Shabnam Sinha. Co-organised with Shikshagraha and Mantra4Change.
- **InvokED 2027 (next):** 26–27 February 2027 (per homepage reference).

Sources: [thewire.in](https://m.thewire.in/article/ptiprnews/invoked-2026-brings-global-and-grassroots-leaders-together-as-shikshagraha-announces-2026-awardees), [shikshalokam.org](https://shikshalokam.org/), [shikshalokam.org/invoked](https://shikshalokam.org/invoked/)

### ELEVATE
Technology platform: **Energising Leadership with Enhanced Visibility and Administrative Transformation towards Excellence.** Open-source microservices for observation, mentoring, analysis, improvement projects.
- Built on **Sunbird** (EkStep Foundation's open-source building blocks)
- All new capabilities contributed back to Sunbird as OSS components
- Hosted on / interoperable with **DIKSHA** (national platform; same Sunbird stack)

Source: [shikshalokam.org](https://shikshalokam.org/), search-derived material from Sunbird / DIKSHA documentation.
