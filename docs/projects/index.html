How a session lands here

Brainstorm → page → push → review.

The per-session loop

  1. Sonal opens Claude Code with a project in mind (named or implied).
  2. Claude routes to the matching workspace and reads the current page state.
  3. Claude enriches the page in place — adds a section, refines a story, drafts a caption, advances a status.
  4. Claude reports back: section title + one line on what changed + "push?"
  5. Sonal confirms → SessionEnd hook commits + pushes → this site rebuilds in ~60 seconds.
  6. Sonal reviews the live URL. She can copy any section title back into chat as "use this" or "not my voice" feedback — the brain learns from both.
